## Local Setup

Clone the repository:

```bash
git clone https://github.com/Harsh04-Vard/harsh-portfolio.git
```

Move into the project directory:

```bash
cd harsh-portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
